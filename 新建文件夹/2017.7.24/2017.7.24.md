七、响应式
    根据屏幕的宽度做出实时的响应
    @media screen and (max-width: 920px){
            .box{
            background:green;
    }
    }当作用在屏幕上且屏幕尺寸小于920px时背景颜色变成绿色
      @media screen and (max-width: 800px) {
                .box{
                    background: yellow;
                }
            }当作用在屏幕上且屏幕尺寸小于800px时背景颜色变成蓝色
    写的时候大的要先写，小的后写。否则小的会被大的覆盖。
